# Clipboard-PHP

一个基于 PHP + SQLite的轻量级云剪切板系统，支持多设备同步文本与图片，并提供 API + PWA 支持。

## iOS快捷指令使用方案：
- 上传iOS文本剪切板到云：https://www.icloud.com/shortcuts/67a17f413a794caab014e6b357a46382


------------------------------------------------------------------------

## ✨ 功能特性

- 用户登录系统
- API Token 鉴权
- 云剪切板（文本 / 图片）
- 支持 Base64 图片同步
- 支持浏览器剪切板自动读取
- PWA 手机端支持

------------------------------------------------------------------------

## 🚀 安装说明

1. 上传项目到服务器
2. 访问 /install.php 初始化数据库
3. 创建管理员账号
4. 删除 install.php
5. 建议修改db.php的数据库信息保证安全

------------------------------------------------------------------------

## 🧩 首页功能（index.php）

功能包括：

### 1️⃣ 云剪切板查看

- 显示当前内容（文本 / 图片）
- 图片以 Base64 渲染展示
- 支持复制文本到本机剪切板

### 2️⃣ 手动更新

- 支持手动输入文本提交
- 支持粘贴图片自动转 Base64 上传

### 3️⃣ 图片大小限制（可自定义）

在代码中：

```php
$maxImageSizeMB = 10;
$maxTextLength  = 10000;
```

可自行调整

------------------------------------------------------------------------

## 📱 PWA 功能（pwa.php）

（移动端优化页面）

### 功能：

- 一键从系统剪切板读取并同步
- 支持图片 Base64 自动上传
- 支持文本同步
- 适合手机端使用

------------------------------------------------------------------------

## 🔌 API 接口：

### 📤 上传剪切板

POST /api_set_clipboard.php

Header: X-API-TOKEN: your_token

参数：

- content_type: text / image
- content: 内容
- mime_type: 可选

### 📥 获取剪切板

GET /api_get_clipboard.php?token=your_token

返回：

- content_type
- content
- mime_type
- updated_at

------------------------------------------------------------------------

## 🔐 Token 机制

API 仅通过 token 认证：

- 支持启用 / 禁用
- 支持过期时间
- 自动记录调用次数
